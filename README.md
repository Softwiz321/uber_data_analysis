# Uber Analytics & Performance Dashboard

An end-to-end data analysis and visualization workspace mapping rideshare metrics, vehicle segmentation, cancellation distribution, and customer-to-driver rating configurations. This repository couples the baseline data profile with an interactive multi-slide Power BI workspace built on corporate visual styling rules.

---

## 📊 Project Overview

This dashboard breaks down overall rideshare operations into tactical slices, focusing on revenue streams, fulfillment roadblocks, and satisfaction trends across a variety of transit options (ranging from standard Sedans and UberXL to specialized micromobility alternatives like Auto, Bike, and eBike).

### Core Components
* **Multi-Page Power BI Report:** Structured visual story breakdown tracking Overall performance, Vehicle Type breakdowns, Revenue leaks, Cancellations, and dual-axis Ratings.
* **Pre-configured Layout Design:** Optimized parameters for multi-tier visualizations ensuring auto-wrapping, crisp dotted gridlines, and highly visible metric displays.
* **Isolated UI Resource Files:** Embedded assets tracking cleaned visual slides and media layouts directly accessible to developers.
